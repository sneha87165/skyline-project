# SkyLine Project

A simple landing page for a travel booking experience built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Main page markup and navigation.
- `style.css` - Custom styling for the hero section, navbar, cards, and responsive layout.
- `script.js` - Interactive behavior for the mobile dropdown menu, passenger counter popup, and trip type toggles.

## Features

- Fixed top navbar with a logo-enhanced brand.
- Hero section with interactive booking search form.
- Smooth dropdown navigation for mobile.
- Passenger selection popup with counters for adults, children, and infants.
- Clean, modern travel-themed styling using a background hero image.

## Usage

1. Open `index.html` in a browser.
2. Use the search form to explore layout and interaction.

## Notes

- The project uses Bootstrap 5 for responsive utilities and Bootstrap Icons for the logo icon.
- No build step is required; this is a static frontend project.
